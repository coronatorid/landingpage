# Landing Page for Coronator

Coronator landing page and crew site for administrational purposes.

### Install packages

```bash
yarn install
or
npm install
```

### Run in development server

```bash
yarn dev
or
npm run dev
```

### Build and serve in production
```bash
> yarn build
or
> npm run build
> yarn start
or
> npm run start
```
